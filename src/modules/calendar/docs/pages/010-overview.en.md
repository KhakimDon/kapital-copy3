# Calendar

**Calendar** is the company-wide calendar, Apple Calendar style. Open it via the **Calendar** item in the left menu.

## Views

The buttons at the top switch between **Day**, **Week**, **Month** and **Year** views. The Year view shows 12 mini-months — dots mean there are events on that day. The **Today** button returns to the current day.

## Calendars

You can keep several colored calendars — for example "Work", "Personal", "Team". Each one has a colored checkbox in the list on the left panel:

1. Click the checkbox to turn a calendar on or off — its events are shown or hidden.
2. Use the **+** (New calendar) button to create a new calendar, pick a name and a color.
3. Click a calendar's name to edit or delete it.

## Creating an event

1. Click **+** at the top — or **double-click** an empty spot in the calendar: the day and hour are picked automatically.
2. Type a title, choose the calendar, set the start and end time. For a full-day event, check **All day**.
3. If needed, choose a **Repeat** — daily, weekly, monthly or yearly — and set the "until…" date. A recurring event appears automatically in every period.
4. Click **Save**.

To change an event, click it — the same dialog opens: edit and save, or delete it.

## Tasks in the calendar

Tasks with due dates from the [Tasks](page:board) board also show up in the calendar — they appear as a chip on the due day; completed ones are marked with a ✓ and struck through. In the **Tasks** section on the left panel you can toggle them per project. Clicking a chip opens the task card.
