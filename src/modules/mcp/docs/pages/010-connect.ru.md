# Подключение AI (MCP)

**MCP-сервер AIBA** подключает вашего любимого AI-помощника — **Claude**, **ChatGPT** или **Gemini** — напрямую к вашим данным в AIBA. Дальше всё через обычный чат: «какие у меня задачи на сегодня?», «найди файл договора аренды», «в каком статусе налоговые отчёты за июль?» — AI выполняет это от вашего имени.

## Получаем токен

1. Кликните свой аватар справа сверху → **MCP**.
2. Кнопкой **Generate** создайте персональный токен.
3. Скопируйте появившийся **Connector URL**. Он выглядит так:
   `https://next.aiba.uz/api/v2/mcp/t/<token>`

## Подключаем Claude

1. В Claude откройте *Settings → Connectors → **Add custom connector***.
2. Введите название (например, «AIBA») и вставьте скопированный адрес в поле URL.
3. Поля OAuth (Client ID / Client Secret) **оставьте пустыми** — токен уже внутри URL.
4. Нажмите **Add**. Теперь можно спрашивать про AIBA прямо в чате.

## Подключаем ChatGPT и Gemini

Работает тот же URL:

- **ChatGPT**: в разделе *Settings → Connectors* добавьте новый connector, вставьте URL, поля аутентификации оставьте пустыми.
- **Gemini**: в настройках откройте раздел добавления MCP-сервера / connector'а, вставьте URL, дополнительная аутентификация не нужна.

AI действует только **в рамках ваших прав** — он не сможет сделать то, что не разрешено вам.

> **Connector URL — это секрет.** Любой, у кого есть этот URL, действует в AIBA от вашего имени. Если он утёк — нажмите **Revoke** на той же странице MCP, и старый URL сразу перестанет работать.
