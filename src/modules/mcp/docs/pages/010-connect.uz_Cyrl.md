# AI'га уланиш (MCP)

**AIBA MCP сервери** севимли AI ёрдамчингизни — **Claude**, **ChatGPT** ёки **Gemini** — тўғридан-тўғри AIBA маълумотларингизга улайди. Кейин ҳаммаси оддий чат орқали: «бугунги вазифаларим қандай?», «ижара шартномаси файлини топ», «июль солиқ ҳисоботлари қай ҳолатда?» — AI сиз номингиздан бажаради.

## Токенни олиш

1. Юқори ўнгдаги профил расмингизни босинг → **MCP**.
2. **Generate** тугмаси билан шахсий токенингизни яратинг.
3. Чиққан **Connector URL** ни нусхаланг. У шундай кўринишда бўлади:
   `https://next.aiba.uz/api/v2/mcp/t/<token>`

## Claude'га улаш

1. Claude'да *Settings → Connectors → **Add custom connector*** ни очинг.
2. Ном ёзинг (масалан, «AIBA») ва URL майдонига нусхалаган манзилни қўйинг.
3. OAuth (Client ID / Client Secret) майдонларини **бўш қолдиринг** — токен URL'нинг ўзида.
4. **Add** ни босинг. Энди чатда AIBA ҳақида сўрашингиз мумкин.

## ChatGPT ва Gemini'га улаш

Худди шу URL ишлайди:

- **ChatGPT**: *Settings → Connectors* бўлимида янги connector қўшинг, URL ни қўйинг, аутентификация майдонларини бўш қолдиринг.
- **Gemini**: созламаларда MCP server / connector қўшиш бўлимини очинг, URL ни қўйинг, қўшимча аутентификация талаб қилинмайди.

AI фақат **сизнинг ҳуқуқларингиз** доирасида ишлайди — сизга рухсат берилмаган нарсани у ҳам қилолмайди.

> **Connector URL — сир.** Бу URL кимда бўлса, у AIBA'да сиз номингиздан иш қилади. Тарқалиб кетса, ўша MCP саҳифасида токенни **Revoke** қилинг — эски URL дарҳол ишламай қолади.
