# Messenger

**Messenger** is the team's internal messaging. Here you chat with colleagues one-on-one or create groups — all inside one company, in real time. Open it from the **Messenger** item in the left menu.

## Start a chat

1. Press the pencil button (FAB) in the bottom-right corner of the list on the left.
2. On the **New chat** tab, search for and pick a user — a direct message (DM) opens. If the chat already exists, it opens instead.
3. On the **New group** tab, name the group, search and select members, and press **Create group**.

Use the search field at the top to find existing chats by name or user.

## Send a message

1. Open a chat and type into the field at the bottom.
2. **Enter** sends the message, **Shift+Enter** starts a new line.
3. Use the smiley button to add emoji.

## Files and images

Press the paperclip button to attach a file or image. You can send several files at once — images show inline in the chat, other files arrive as downloads.

## Voice message

1. When the text field is empty, a microphone button sits on the right. Press it — recording starts.
2. To cancel the recording, press the trash button.
3. To send it, press the check button.

## Message actions

Hover over a message to open its menu:

- **Reply** — reply quoting that message;
- **Edit** — change the text of your own message;
- **Copy** — copy the text;
- **Delete** — delete the message.

## Managing a group

Press the title at the top of the chat or the info button to open the **Info** panel on the right. There you can:

- see the group members and their roles — **owner** and **admin**;
- press **Add** to search for and add a new member;
- an admin or owner can remove a member.

## Mute

The **Notifications** switch in the info panel mutes the chat. A muted chat is marked with a bell icon in the list.

## Counters and "typing…"

- Unread messages show as a number next to the chat (over 99 shows `99+`).
- When someone is typing, **"typing…"** appears in the list and above the chat.

> On a phone the list and the chat swap on one screen: opening a chat expands it full-screen, and the back button returns to the list.
