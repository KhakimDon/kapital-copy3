# Messenger

**Messenger** — жамоа учун ички хабар алмашиш. Бу ерда ҳамкасблар билан шахсий ёзишасиз ёки гуруҳлар тузасиз — ҳаммаси битта корхона ичида, реал вақтда. Чап менюдаги **Messenger** бандидан очилади.

## Суҳбат бошлаш

1. Чапдаги рўйхатнинг пастки ўнг бурчагидаги қалам тугмасини (FAB) босинг.
2. **Янги суҳбат** бўлимида фойдаланувчини қидиринг ва танланг — у билан шахсий суҳбат (DM) очилади. Агар суҳбат аллақачон бор бўлса, у очилади.
3. **Янги гуруҳ** бўлимида гуруҳга ном беринг, аъзоларни қидириб белгиланг ва **Гуруҳ яратиш** ни босинг.

Юқоридаги қидирув майдони орқали мавжуд суҳбатларни ном ёки фойдаланувчи бўйича топасиз.

## Хабар ёзиш

1. Суҳбатни очинг ва пастдаги майдонга матн ёзинг.
2. **Enter** хабарни юборади, **Shift+Enter** янги қатордан бошлайди.
3. Кулгичлар тугмаси билан эможи қўшасиз.

## Файл ва расмлар

Скрепка (қисқич) тугмасини босиб файл ёки расм бириктиринг. Бир вақтда бир нечта файл юбориш мумкин — расмлар суҳбатда кўринади, бошқа файллар юклаб олинадиган бўлиб чиқади.

## Овозли хабар

1. Матн майдони бўш бўлса, ўнг томонда микрофон тугмаси туради. Уни босинг — ёзиб олиш бошланади.
2. Ёзиб олишни бекор қилиш учун саватча тугмасини босинг.
3. Юбориш учун тасдиқ (белги) тугмасини босинг.

## Хабар устидаги амаллар

Хабар устига борсангиз меню чиқади:

- **Жавоб бериш** — ўша хабарга иқтибос қилиб жавоб ёзасиз;
- **Таҳрирлаш** — ўз хабарингиз матнини ўзгартирасиз;
- **Нусхалаш** — матнни нусха оласиз;
- **Ўчириш** — хабарни ўчирасиз.

## Гуруҳни бошқариш

Суҳбат тепасидаги номни ёки маълумот тугмасини босиб ўнг панелдаги **Маълумот** ни очинг. У ерда:

- гуруҳ аъзолари рўйхати ва роллари — **эгаси** ва **админ**;
- **Қўшиш** тугмаси билан янги аъзо қидириб қўшасиз;
- админ ва эгаси аъзони рўйхатдан чиқара олади.

## Овозсиз қилиш (mute)

Маълумот панелидаги **Билдиришномалар** тумблери суҳбатни овозсиз қилади. Овозсиз суҳбат рўйхатда қўнғироқ белгиси билан белгиланади.

## Ҳисоблагичлар ва «ёзмоқда…»

- Ўқилмаган хабарлар суҳбат ёнида рақамли белги бўлиб туради (99 дан кўп бўлса `99+`).
- Кимдир ёзаётганда рўйхатда ва суҳбат тепасида **«ёзмоқда…»** кўрсатилади.

> Мобил қурилмада рўйхат ва суҳбат бир экранда алмашади: суҳбатни очсангиз тўлиқ очилади, орқага тугмаси рўйхатга қайтаради.
