# Открываем и заполняем карточку

Клик по карточке открывает подробное окно:

![Окно карточки](img/card.png)

- **Описание** — кликните по тексту и пишите: жирный, курсив, списки работают.
- **Файлы и картинки** — перетащите файл в окно или выберите через кнопку. Картинку можно сделать **обложкой** (cover) — карточка на доске будет с изображением.
- **Сроки** — задайте дату начала и дедлайн. О приближающихся и просроченных задачах придёт напоминание.
- **Метки** — для тематической разметки (например: `backend`, `дизайн`).

> Изменения в карточке уведомляют участников — см. [Уведомления](page:notifications).
