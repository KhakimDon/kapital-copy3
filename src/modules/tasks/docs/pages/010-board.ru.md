# Знакомство с доской

Этот модуль — доска задач вашей команды (как Jira/Trello). Каждая работа — это **карточка**, карточки движутся по **колонкам** (этапам) слева направо: *Новые → В работе → Проверка → Готово*.

> Модуль открывается из левого меню — **Задачи**. Сначала выберите компанию сверху — у каждой компании своя доска.

![Доска задач](img/board.png)

Что на экране:

- **Колонки** — этапы работы. Над каждой колонкой — название, количество карточек и процент.
- **Карточки** — задачи. На карточке: номер (например `DEV-12`), заголовок, фото ответственного, признак приоритета и метки.
- **Значок 🔥** — задача с приоритетом *высокий* или *срочно* «горит».
- Сверху: **переключатель вида** (Доска / Список), **поиск**, **группировка** и **фильтры**.
