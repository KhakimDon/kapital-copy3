# Картани очиш ва тўлдириш

Картани боссангиз батафсил ойна очилади:

![Карта ойнаси](img/card.png)

- **Тавсиф** — матнни босинг ва ёзинг: қалин, курсив, рўйхатлар ишлайди.
- **Расм/файл бириктириш** — файлни ойнага судраб ташланг ёки бириктириш тугмасидан танланг. Расм бириктирсангиз уни картанинг **муқоваси** (cover) қилиб қўйишингиз мумкин — карта доскада расм билан кўринади.
- **Муддат** — бошланиш ва тугаш санасини белгиланг. Муддати яқин ёки ўтган вазифалар сизга эслатма юборади.
- **Ёрлиқлар** — вазифаларни мавзу бўйича белгилаш учун (масалан: `backend`, `дизайн`).

> Картадаги ўзгаришлар алоқадорларга хабар беради — қаранг: [Билдиришномалар](page:notifications).
