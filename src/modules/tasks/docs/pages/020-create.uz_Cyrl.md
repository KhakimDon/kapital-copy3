# Биринчи вазифани яратиш

1. Керакли устун остидаги **+ Вазифа қўшиш** тугмасини босинг.
2. Сарлавҳани ёзинг ва **Enter** ни босинг — карта шу устунда пайдо бўлади.
3. Батафсил маълумот учун картани босиб очинг.

![Тезкор қўшиш](img/quick-add.png)

> Вазифа номи қисқа ва аниқ ҳаракатни билдирсин: «Логин саҳифасидаги хатони тузатиш» — яхши; «муаммо» — ёмон.

> Маслаҳат: яратиш пайтидаёқ масъул тайинлаш мумкин — батафсил: [Масъул тайинлаш](page:assign).
