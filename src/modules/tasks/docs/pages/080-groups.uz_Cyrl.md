# Гуруҳлаш ва қидирув

- **Гуруҳлар** тугмаси доскани горизонтал қаторларга бўлади: эпик, масъул ёки муҳимлик бўйича — катта доскада жуда қулай.
- **Қидирув** (юқоридаги лупа) — ном бўйича тезкор қидирув ойнаси очилади: ёзинг, стрелкалар билан танланг, **Enter** билан очинг.
- Бажарилган эски карталар доскани тўлдирмаслиги учун маълум кундан кейин автоматик яширилади (созламалардан бошқарилади).
