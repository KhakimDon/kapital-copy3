# Подзадачи (subtask) и Эпики

- **Подзадача** — способ разбить большую задачу на части: внутри карточки создаёте список подзадач, их прогресс виден на родительской карточке. Когда подзадача завершается, владельцы родительской задачи получают уведомление.
- **Эпик** — крупная тема, объединяющая несколько задач (например «Мобильное приложение v2»). Эпик можно привязывать к карточкам из разных проектов; на карточке эпик отображается цветной меткой.
