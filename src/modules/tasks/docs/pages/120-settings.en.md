# Project settings

The project owner/admin configures:

- the project **name, key** (card numbers come from it: `DEV-1`, `DEV-2`…), description and **cover image**;
- **columns**: add, rename, colors, order, and "who may move cards into this column" restrictions;
- **access**: open to everyone or only to selected members;
- the default assignee and default reporter.
