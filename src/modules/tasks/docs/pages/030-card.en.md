# Open and fill in a card

Clicking a card opens the detail panel:

![The card panel](img/card.png)

- **Description** — click the text and type: bold, italics and lists work.
- **Files & images** — drag a file onto the panel or use the attach button. An image can become the card's **cover** — the card shows the picture on the board.
- **Dates** — set a start date and a due date. You'll get reminders for tasks that are due soon or overdue.
- **Labels** — tag tasks by topic (e.g. `backend`, `design`).

> Card changes notify the people involved — see [Notifications](page:notifications).
