# Connect an AI assistant (MCP)

You can also work with tasks through **Claude** (or another AI client): "show my tasks for today", "create 'Prepare the report' in DEV and assign Ali" — all via chat.

![The MCP connect page](img/mcp.png)

To connect:

1. Click your avatar in the top-right → **MCP**.
2. Press **Generate** to get a personal connection token (read-write — full access, read-only — reading only).
3. Copy the **Connector URL** and paste it into Claude: *Settings → Connectors → Add custom connector*.
4. Claude now reads and manages tasks on your behalf — but it **can never do more than you are allowed to**.

> The token is like a password. If it leaks, hit **Revoke** on the MCP page — the old URL dies instantly.
