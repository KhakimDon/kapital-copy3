# Масъул тайинлаш

1. Картани очинг ва **Масъул** майдонини босинг, жамоадан одам танланг.
2. Ёки тезроқ: картанинг устига **сичқончанинг ўнг тугмаси** билан босинг → **Масъул** → одамни танланг.

Тайинланган одамга дарҳол **билдиришнома** боради. Битта вазифага бир нечта масъул қўшса бўлади.

> Ўнг тугма менюси — энг тез йўл: очиш, муҳимлик, масъул, устунга кўчириш, кузатиш, нусхалаш ва ўчириш — ҳаммаси битта жойда.
