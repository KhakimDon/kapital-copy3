# Move a task to the next stage

- **Drag** the card to another column with the mouse;
- or right-click → **Move to column** → pick the column.

When a card lands in a *Done*-type column the task counts as **completed** and participants are notified. Moving into some columns may be restricted to certain roles — then the action is unavailable with an explanation.
