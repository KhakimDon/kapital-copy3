# Birinchi vazifani yaratish

1. Kerakli ustun ostidagi **+ Vazifa qo'shish** tugmasini bosing.
2. Sarlavhani yozing va **Enter** ni bosing — karta shu ustunda paydo bo'ladi.
3. Batafsil ma'lumot uchun kartani bosib oching.

![Tezkor qo'shish](img/quick-add.png)

> Vazifa nomi qisqa va aniq harakatni bildirsin: "Login sahifasidagi xatoni tuzatish" — yaxshi; "muammo" — yomon.

> Maslahat: yaratish paytidayoq mas'ul tayinlash mumkin — batafsil: [Mas'ul tayinlash](page:assign).
