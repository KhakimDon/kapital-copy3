# Группировка и поиск

- Кнопка **Группы** делит доску на горизонтальные ряды: по эпику, ответственному или приоритету — удобно на большой доске.
- **Поиск** (лупа сверху) открывает окно быстрого поиска: печатайте, выбирайте стрелками, открывайте **Enter**.
- Старые завершённые карточки автоматически скрываются через заданное число дней (настраивается), чтобы не захламлять доску.
