# Создаём первую задачу

1. Нажмите **+ Добавить задачу** под нужной колонкой.
2. Введите заголовок и нажмите **Enter** — карточка появится в этой колонке.
3. Для подробностей откройте карточку кликом.

![Быстрое добавление](img/quick-add.png)

> Название должно означать короткое и понятное действие: «Исправить ошибку на странице входа» — хорошо; «проблема» — плохо.

> Совет: ответственного можно указать прямо при создании — подробнее: [Назначение ответственного](page:assign).
