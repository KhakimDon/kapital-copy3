# AI yordamchini ulash (MCP)

Vazifalar bilan **Claude** (yoki boshqa AI mijoz) orqali ham ishlash mumkin: "menga bugungi vazifalarimni ko'rsat", "DEV proyektida 'Hisobot tayyorlash' vazifasini yarat, Alini mas'ul qil" — hammasi chat orqali.

![MCP ulash sahifasi](img/mcp.png)

Ulash:

1. Yuqori o'ngdagi profil rasmingizni bosing → **MCP**.
2. **Generate** bilan shaxsiy ulanish tokenini oling (read-write — to'liq, read-only — faqat o'qish).
3. Chiqqan **Connector URL** ni nusxalang va Claude'da *Settings → Connectors → Add custom connector* ga qo'ying.
4. Endi Claude sizning nomingizdan vazifalarni o'qiydi va boshqaradi — lekin **sizning huquqlaringizdan ortiq hech narsa qilolmaydi**.

> Token — parol kabi maxfiy. Kimgadir tarqalib ketsa, MCP sahifasidan **Revoke** qiling — eski URL darhol o'chadi.
