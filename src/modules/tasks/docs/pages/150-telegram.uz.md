# Telegram xabarnomalar

Doskadagi har bir hodisa — vazifa **yaratildi**, **ko'chirildi**, **bajarildi**, **mas'ul tayinlandi**, **izoh yozildi**, **o'zgartirildi** yoki **o'chirildi** — Telegram **guruhingizga** avtomatik xabar bo'lib boradi. Jamoa doskani ochmasdan ham nimalar bo'layotganini ko'rib turadi.

Ulash:

1. Tenant admini [@BotFather](https://t.me/BotFather) orqali bot yaratadi va uning tokenini *Proyekt sozlamalari → **Telegram xabarnomalar** → **Botlar*** bo'limiga qo'shadi. Tokenni faqat admin qo'sha oladi.
2. Botni Telegram guruhingizga a'zo qilib qo'shing.
3. Guruhning **chat ID** sini oling — masalan, guruhga [@getidsbot](https://t.me/getidsbot) ni qo'shib. Guruh ID si `-100` bilan boshlanadi.
4. Proyekt sozlamalarida botni tanlang, chat ID ni kiriting, qaysi hodisalar yuborilishini belgilang, **Test** tugmasi bilan tekshirib ko'ring va **Saqlash** ni bosing.

Bitta botni bir nechta proyektga ulash mumkin — har bir proyekt o'z guruhiga yozadi.

> Xabarnomalar ilova ichida qilingan o'zgarishlar uchun ham, [AI (MCP) orqali](page:connect) qilingan o'zgarishlar uchun ham yuboriladi.
