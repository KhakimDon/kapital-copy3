# Vazifani keyingi bosqichga o'tkazish

- Kartani sichqoncha bilan **sudrab** boshqa ustunga tashlang;
- yoki o'ng tugma → **Ustunga ko'chirish** → ustunni tanlang.

Karta *Bajarildi* turidagi ustunga tushsa, vazifa **yakunlangan** hisoblanadi va aloqadorlarga xabar boradi. Ba'zi ustunlarga ko'chirish faqat ma'lum rollarga ruxsat etilgan bo'lishi mumkin — bunda tugma ishlamaydi va sabab ko'rsatiladi.
