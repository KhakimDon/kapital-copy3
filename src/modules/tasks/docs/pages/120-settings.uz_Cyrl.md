# Проект созламалари

Проект эгаси/админ қуйидагиларни созлайди:

- проект **номи, калити** (карталар рақами шу калитдан: `DEV-1`, `DEV-2`…), тавсифи ва **муқова расми**;
- **устунлар**: қўшиш, номлаш, ранг, тартиб ва «ким бу устунга кўчира олади» чеклови;
- **кириш ҳуқуқи**: проект очиқ ёки фақат танланган аъзоларга;
- стандарт масъул ва стандарт муаллиф.
