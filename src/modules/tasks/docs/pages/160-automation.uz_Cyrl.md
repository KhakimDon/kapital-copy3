# Автоматлаштириш

Доска такрорланадиган ишларни ўзи бажара олади. **Автоматлаштириш** — код ёзмасдан тузиладиган қоидалар. Улар *[Проект созламалари](page:settings) → **Автоматлаштириш*** бўлимида туради; қоидаларни проект эгаси/админ тузади.

Ҳар бир қоида уч қисмдан иборат:

- **ҚАЧОН** (триггер) — қоида қайси ҳодисада ишга тушади;
- **АГАР** (шартлар) — ихтиёрий фильтрлар;
- **УНДА** (амаллар) — нима қилиниши керак.

## Қоида яратиш

1. Проект созламаларида **Автоматлаштириш** бўлимини топинг ва **Қоида** тугмасини босинг.
2. Қоида номини ёзинг.
3. **Қачон** — триггерни танланг:
   - **Яратилганда** — янги вазифа пайдо бўлганда (ихтиёрий: қайси устунда);
   - **Кўчирилганда** — карта устундан устунга ўтганда; **Қаердан** ва **Қаерга** устунларини белгилаш мумкин (ёки «Ҳар қандай»);
   - **Тайинланганда** — масъул ўзгарганда;
   - **Муҳимлик ўзгарганда** — ихтиёрий: қайси янги муҳимликда;
   - **Изоҳ ёзилганда** — вазифага изоҳ қўшилганда.
4. **Агар** — керак бўлса шарт қўшинг: майдон (**Муҳимлик**, **Устун**, **Тури**, **Масъул**, **Ёрлиқ**, **Сарлавҳа**) + шарт (**тенг**, **тенг эмас**, **бўш**, **бўш эмас**, **ўз ичига олади**) + қиймат. Бир нечта шарт бўлса, ҳаммаси бажарилиши керак.
5. **Унда** — бир ёки бир нечта амал қўшинг:
   - **Устунга кўчириш** — картани танланган устунга ўтказади;
   - **Масъул тайинлаш** — танланган фойдаланувчига, **Муаллифга** ёки **Олиб ташлаш**;
   - **Муҳимликни ўзгартириш**;
   - **Ёрлиқ қўшиш** / **Ёрлиқни олиб ташлаш**;
   - **Муддатни суриш** — муддатни кўрсатилган кунга суради;
   - **Изоҳ ёзиш** — шаблон бўйича автоматик изоҳ;
   - **Кузатувчиларга хабар** — кузатувчиларга [билдиришнома](page:notifications) юборади;
   - **Telegram хабари** — [уланган Telegram гуруҳига](page:telegram) шаблонли хабар.
6. **Сақлаш** ни босинг.

Изоҳ ва Telegram шаблонларида ўринбосарлар ишлайди: `{{title}}` — вазифа сарлавҳаси, `{{actor}}` — амални бажарган фойдаланувчи, `{{priority}}` — муҳимлик.

## Ёқиш ва ўчириш

Рўйхатдаги ҳар бир қоиданинг чап томонида тумблер бор — қоидани ўчирмасдан вақтинча тўхтатиб қўйиш мумкин. Қалам белгиси таҳрирлайди, саватча ўчиради.

## Мисоллар

- **«Done»га кўчирилганда** → **Изоҳ ёзиш**: `{{actor}} вазифани якунлади` + **Telegram хабари**.
- **Яратилганда**, агар **Масъул бўш** → **Муаллифга** тайинлаш.
- **Муҳимлик ўзгарганда** (янги муҳимлик: Шошилинч) → **Ёрлиқ қўшиш**: `urgent` + **Кузатувчиларга хабар**.

> Қоидалар илова ичидаги ўзгаришларда ҳам, [AI (MCP) орқали](page:connect) қилинган ўзгаришларда ҳам ишлайди. Бир қоиданинг амали бошқа қоидани ишга тушириши мумкин — занжир кўпи билан 3 қадам давом этади.
