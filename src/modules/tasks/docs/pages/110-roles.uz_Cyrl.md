# Роллар ва рухсатлар

Ҳар бир фойдаланувчининг проектда роли бор: **Эгаси · Аъзо · Кузатувчи** (админ махсус роллар ҳам яратиши мумкин). Рол нималарга рухсат беришини белгилайди: кўриш, яратиш, таҳрирлаш, кўчириш, ўчириш, бошқариш. Бирор тугма ишламаса — катта эҳтимол ролингиз бунга рухсат бермайди; проект эгасига мурожаат қилинг.
