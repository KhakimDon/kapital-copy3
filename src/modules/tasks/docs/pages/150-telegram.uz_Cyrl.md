# Telegram хабарномалар

Доскадаги ҳар бир ҳодиса — вазифа **яратилди**, **кўчирилди**, **бажарилди**, **масъул тайинланди**, **изоҳ ёзилди**, **ўзгартирилди** ёки **ўчирилди** — Telegram **гуруҳингизга** автоматик хабар бўлиб боради. Жамоа доскани очмасдан ҳам нималар бўлаётганини кўриб туради.

Улаш:

1. Тенант админи [@BotFather](https://t.me/BotFather) орқали бот яратади ва унинг токенини *Проект созламалари → **Telegram хабарномалар** → **Ботлар*** бўлимига қўшади. Токенни фақат админ қўша олади.
2. Ботни Telegram гуруҳингизга аъзо қилиб қўшинг.
3. Гуруҳнинг **chat ID** сини олинг — масалан, гуруҳга [@getidsbot](https://t.me/getidsbot) ни қўшиб. Гуруҳ ID си `-100` билан бошланади.
4. Проект созламаларида ботни танланг, chat ID ни киритинг, қайси ҳодисалар юборилишини белгиланг, **Test** тугмаси билан текшириб кўринг ва **Сақлаш** ни босинг.

Битта ботни бир нечта проектга улаш мумкин — ҳар бир проект ўз гуруҳига ёзади.

> Хабарномалар илова ичида қилинган ўзгаришлар учун ҳам, [AI (MCP) орқали](page:connect) қилинган ўзгаришлар учун ҳам юборилади.
