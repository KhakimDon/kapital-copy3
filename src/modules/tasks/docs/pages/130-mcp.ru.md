# Подключаем AI-помощника (MCP)

С задачами можно работать и через **Claude** (или другой AI-клиент): «покажи мои задачи на сегодня», «создай в проекте DEV задачу „Подготовить отчёт“ и назначь Али» — всё через чат.

![Страница подключения MCP](img/mcp.png)

Подключение:

1. Кликните свой аватар справа сверху → **MCP**.
2. Кнопкой **Generate** получите персональный токен (read-write — полный доступ, read-only — только чтение).
3. Скопируйте **Connector URL** и вставьте в Claude: *Settings → Connectors → Add custom connector*.
4. Теперь Claude читает и ведёт задачи от вашего имени — но **не может сделать больше, чем позволено вам**.

> Токен — как пароль. Если он попал к посторонним, нажмите **Revoke** на странице MCP — старая ссылка сразу перестанет работать.
