# AI ёрдамчини улаш (MCP)

Вазифалар билан **Claude** (ёки бошқа AI мижоз) орқали ҳам ишлаш мумкин: «менга бугунги вазифаларимни кўрсат», «DEV проектида "Ҳисобот тайёрлаш" вазифасини ярат, Алини масъул қил» — ҳаммаси чат орқали.

![MCP улаш саҳифаси](img/mcp.png)

Улаш:

1. Юқори ўнгдаги профил расмингизни босинг → **MCP**.
2. **Generate** билан шахсий уланиш токенини олинг (read-write — тўлиқ, read-only — фақат ўқиш).
3. Чиққан **Connector URL** ни нусхаланг ва Claude'да *Settings → Connectors → Add custom connector* га қўйинг.
4. Энди Claude сизнинг номингиздан вазифаларни ўқийди ва бошқаради — лекин **сизнинг ҳуқуқларингиздан ортиқ ҳеч нарса қилолмайди**.

> Токен — парол каби махфий. Кимгадир тарқалиб кетса, MCP саҳифасидан **Revoke** қилинг — эски URL дарҳол ўчади.
