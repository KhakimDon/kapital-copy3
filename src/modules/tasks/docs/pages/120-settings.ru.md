# Настройки проекта

Владелец/админ проекта настраивает:

- **название, ключ** (номера карточек из этого ключа: `DEV-1`, `DEV-2`…), описание и **обложку** проекта;
- **колонки**: добавление, названия, цвета, порядок и ограничение «кто может переносить в эту колонку»;
- **доступ**: проект открытый или только для выбранных участников;
- ответственного и автора по умолчанию.
