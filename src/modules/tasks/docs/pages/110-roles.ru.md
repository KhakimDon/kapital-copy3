# Роли и права

У каждого пользователя в проекте есть роль: **Владелец · Участник · Наблюдатель** (админ может создавать и свои роли). Роль определяет права: просмотр, создание, редактирование, перемещение, удаление, управление. Если какая-то кнопка не работает — скорее всего, ваша роль этого не позволяет; обратитесь к владельцу проекта.
