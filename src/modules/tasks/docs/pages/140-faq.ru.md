# Частые вопросы

- **Не вижу свою карточку.** Возможно, включён фильтр или поиск; завершённые могли скрыться автоматически.
- **Не могу перенести в колонку.** На колонке стоит ограничение по ролям — спросите владельца проекта.
- **Не пришло уведомление.** О собственных действиях уведомления не приходят (только другим).
- **Удалил задачу.** Удаление необратимо — дважды подумайте, прежде чем удалять важное.
