# Назначаем ответственного

1. Откройте карточку и кликните поле **Ответственный**, выберите человека.
2. Или быстрее: **правый клик** по карточке → **Ответственный** → выберите человека.

Назначенному сразу придёт **уведомление**. У задачи может быть несколько ответственных.

> Меню правой кнопки — самый быстрый путь: открыть, приоритет, ответственный, переместить в колонку, наблюдать, дублировать и удалить — всё в одном месте.
