# Тез-тез сўраладиган саволлар

- **Картам кўринмаяпти.** Фильтр ёки қидирув ёқилган бўлиши мумкин; «Бажарилган»лар автоматик яширинган бўлиши ҳам мумкин.
- **Устунга кўчира олмаяпман.** Устунда рол чеклови бор — проект эгасидан сўранг.
- **Билдиришнома келмади.** Ўзингиз қилган амал учун сизга билдиришнома бормайди (фақат бошқаларга).
- **Вазифани ўчириб юбордим.** Ўчириш қайтарилмайди — муҳим вазифаларни ўчиришдан аввал икки марта ўйланг.
