# Кичик вазифалар (subtask) ва Эпиклар

- **Subtask** — катта вазифани бўлакларга бўлиш: карта ичида кичик вазифалар рўйхатини яратасиз, уларнинг бажарилиши ота-картада кўринади. Subtask якунланса, ота-вазифа эгаларига хабар боради.
- **Эпик** — бир нечта вазифани бирлаштирувчи йирик мавзу (масалан «Мобил илова v2»). Эпик бошқа проектдаги карталарга ҳам уланади; картада эпик номи рангли белги бўлиб туради.
