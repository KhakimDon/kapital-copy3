# Guruhlash va qidiruv

- **Guruhlar** tugmasi doskani gorizontal qatorlarga bo'ladi: epik, mas'ul yoki muhimlik bo'yicha — katta doskada juda qulay.
- **Qidiruv** (yuqoridagi lupa) — nom bo'yicha tezkor qidiruv oynasi ochiladi: yozing, strelkalar bilan tanlang, **Enter** bilan oching.
- Bajarilgan eski kartalar doskani to'ldirmasligi uchun ma'lum kundan keyin avtomatik yashiriladi (sozlamalardan boshqariladi).
