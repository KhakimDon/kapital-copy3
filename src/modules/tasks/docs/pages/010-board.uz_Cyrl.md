# Доска билан танишув

Бу модул — жамоангизнинг вазифалар доскаси (Jira/Trello каби). Ҳар бир иш **карта** кўринишида бўлади, карталар **устунлар** (босқичлар) бўйлаб чапдан ўнгга ҳаракатланади: *Янги → Жараёнда → Текширишда → Бажарилди*.

> Модул чап менюдаги **Вазифалар** бўлимидан очилади. Аввал юқоридан компанияни танланг — ҳар бир компаниянинг ўз доскаси бўлади.

![Вазифалар доскаси](img/board.png)

Экранда нималар бор:

- **Устунлар** — иш босқичлари. Ҳар бир устун тепасида номи, карталар сони ва фоизи кўринади.
- **Карталар** — вазифалар. Картада: вазифа рақами (масалан `DEV-12`), сарлавҳа, масъул одамнинг расми, муҳимлик белгиси ва ёрлиқлар.
- **🔥 белгиси** — муҳимлиги *юқори* ёки *шошилинч* вазифа «ёниб турибди» дегани.
- Юқорида: **кўриниш алмаштиргич** (Доска / Рўйхат), **қидирув**, **гуруҳлаш** ва **фильтрлар**.
