# Grouping and search

- The **Groups** button splits the board into horizontal rows: by epic, assignee or priority — great on a busy board.
- **Search** (the magnifier on top) opens a quick-search dialog: type, pick with arrow keys, open with **Enter**.
- Old completed cards auto-hide after a configurable number of days so the board stays clean.
