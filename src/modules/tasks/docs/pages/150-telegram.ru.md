# Уведомления в Telegram

Каждое событие на доске — задача **создана**, **перемещена**, **завершена**, **назначен ответственный**, **добавлен комментарий**, **изменена** или **удалена** — автоматически уходит сообщением в вашу **группу** Telegram. Команда видит, что происходит, даже не открывая доску.

Подключение:

1. Админ тенанта создаёт бота через [@BotFather](https://t.me/BotFather) и добавляет его токен в *Настройки проекта → **Уведомления Telegram** → **Боты***. Добавить токен может только админ.
2. Добавьте бота участником в вашу Telegram-группу.
3. Получите **chat ID** группы — например, добавив в группу [@getidsbot](https://t.me/getidsbot). ID группы начинается с `-100`.
4. В настройках проекта выберите бота, введите chat ID, отметьте нужные события, проверьте кнопкой **Test** и нажмите **Сохранить**.

Одного бота можно подключить к нескольким проектам — каждый проект пишет в свою группу.

> Уведомления приходят и об изменениях, сделанных в приложении, и об изменениях, сделанных [через AI (MCP)](page:connect).
