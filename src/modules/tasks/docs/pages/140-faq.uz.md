# Tez-tez so'raladigan savollar

- **Kartam ko'rinmayapti.** Filtr yoki qidiruv yoqilgan bo'lishi mumkin; "Bajarilgan"lar avtomatik yashiringan bo'lishi ham mumkin.
- **Ustunga ko'chira olmayapman.** Ustunda rol cheklovi bor — proyekt egasidan so'rang.
- **Bildirishnoma kelmadi.** O'zingiz qilgan amal uchun sizga bildirishnoma bormaydi (faqat boshqalarga).
- **Vazifani o'chirib yubordim.** O'chirish qaytarilmaydi — muhim vazifalarni o'chirishdan avval ikki marta o'ylang.
