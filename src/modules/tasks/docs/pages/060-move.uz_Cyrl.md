# Вазифани кейинги босқичга ўтказиш

- Картани сичқонча билан **судраб** бошқа устунга ташланг;
- ёки ўнг тугма → **Устунга кўчириш** → устунни танланг.

Карта *Бажарилди* туридаги устунга тушса, вазифа **якунланган** ҳисобланади ва алоқадорларга хабар боради. Баъзи устунларга кўчириш фақат маълум ролларга рухсат этилган бўлиши мумкин — бунда тугма ишламайди ва сабаб кўрсатилади.
