# Переводим задачу на следующий этап

- **Перетащите** карточку мышью в другую колонку;
- или правый клик → **Переместить в колонку** → выберите колонку.

Когда карточка попадает в колонку типа *Готово*, задача считается **завершённой**, участники получают уведомление. В некоторые колонки перенос может быть разрешён только определённым ролям — тогда действие будет недоступно с пояснением.
